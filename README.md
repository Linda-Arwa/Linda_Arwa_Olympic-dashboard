# Linda_Arwa_Olympic-dashboard
An interactive dashboard exploring 120 years of Olympic athlete data (1896–2016), built as the bonus task for the Data Analytics & Visualization Take-Home Examination (DS20).

## Features
- Dropdown filter to explore data by Sport
- Age distribution histogram
- Height vs Weight scatter plot
- Medal counts by Sex (Gold, Silver, Bronze)

## Data Source
Olympic athlete data from the "120 Years of Olympic History" dataset (Kaggle), 
originally compiled by sports-reference.com.

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Built With
- Python
- Streamlit
- Pandas
- Matplotlib / Seaborn
